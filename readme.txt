# 🌤 Weather App

A modern weather application built with Vanilla JavaScript.

## 🚀 Features
- Search by city
- Current weather
- 5-day forecast
- Geolocation support
- Dark mode
- LocalStorage

## 🛠 Tech Stack
- HTML
- CSS
- bootstrap
- JavaScript (ES6)
- OpenWeather API
